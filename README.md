# ST Gaming 伺服器檔案手動安裝包

讓你快速安裝/更新 ST Gaming 伺服器所需的檔案，並修復正版 CS1.6 繁體中文顯示英文的問題。

## 下載

前往 [Releases](../../releases) 頁面下載最新版本的 `ST-Gaming-package-x.x.x.exe`

## 安裝方式

1. 下載最新版本的 exe 檔案
2. 將 exe 放到**正版 CS1.6 根目錄**（與 `hl.exe` 相同的資料夾）
3. 執行安裝程式
4. 點擊「開始安裝」
5. 完成！

## 注意事項

- 安裝會覆蓋現有檔案
- **若有自訂噴漆，請先備份再自行還原**
- 僅支援正版 CS1.6
